线性代数

通过学习一些基本的线性代数概念，熟悉其英文表达，以及在Tex中表示方式。

标量、向量、矩阵、张量

  1. 标量(scalar)：一个单独的数。

    • 表示方式：$\mathcal{x}$ (小写变量表示)

  2. 向量(vector)：一列数。

    • 表示方式：$\boldsymbol{x}$ (倾斜粗体小写变量表示)
    • 将向量看成空间中的点，每个元素是不同坐标轴上的坐标
    • 定义集合 $\mathcal{S} = \lbrace 1, 2, 3 \rbrace$, 指定 $\mathcal{x}_1, \mathcal{x}_2,\mathcal{x}_3$，写作$\mathbf{x}_s$
    • 除S外其他元素表示为 $\boldsymbol{x}_{-s}$

  3. 矩阵(matrix)：二维数组。

    • 行(row), 列(column)
    • 表达方式：$\boldsymbol{A}$ （倾斜粗体大写变量名称）

  4. 张量(tensor)：多维数组

    • 表达方式：$\mathbf{A}$ (粗体大写变量名称)

  5. 其他

    • 转置(transpose)
    • 广播(broadcasting)

乘法

  1. 两个矩阵标准乘积，可以写作$\boldsymbol{C} = \boldsymbol{AB}$， 定义为 $\boldsymbol{C}{i,j}=\sum{k}\boldsymbol{A}{i,k}\boldsymbol{B}{k,j}$。

  2. 两个矩阵对应元素乘积(element-wise product)或者叫做 Hadamard乘积(Hadamard product)，记为 $\boldsymbol{A}\odot\boldsymbol{B}$

  3. 向量 $\boldsymbol{x},\boldsymbol{y}$ 的点积(dot product)记为 $\boldsymbol{x}^T\boldsymbol{y}$

  4. 线性方程组表示为 $\boldsymbol{Ax} = \boldsymbol{b}$

单位矩阵和逆矩阵

  1. 逆矩阵(matrix inversion)，记作 $\boldsymbol{A}^{-1}$

  2. 单位矩阵(identity inversion)，n维单位向量记作 $\boldsymbol{I}_n$
